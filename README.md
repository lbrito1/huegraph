# Huegraph

![](demo.gif)

Graphs with [hue](https://en.wikipedia.org/wiki/Hue).

## Installation

    $ gem install huegraph

## Usage

`huegraph` outputs the default 20x20 demo seen above. Check `hello_graph.rb` for other options.
